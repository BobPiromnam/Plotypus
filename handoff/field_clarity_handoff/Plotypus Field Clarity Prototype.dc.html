Field-type rules · for implementation
Value cell (editable)
  • Borderless at rest — unchanged from today (border/background:transparent).
  • Hover → inset 1px #d4ccbd + white fill + text cursor.
  • Focus → inset 2px #3f6b5e (existing focus token).
  • Header label carries a small muted pencil glyph.
Computed cell (Status)
  • Faint persistent column tint #f0eee7 + thin left rule.
  • No hover affordance, cursor:default, not tab-focusable, aria-readonly.
  • Header marked with an Auto lock chip.
  • Pill reuses existing .row-status-badge colours.
